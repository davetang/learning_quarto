Pointer

Pointer Extension

The pointer extension switches your cursor to a pointer style while presenting.

Install:

quarto add quarto-ext/pointer

Enable in YAML:

revealjs-plugins:
  - pointer

Try it now: press q to toggle the pointer on this slide.

The pointer is red by default and 16px in size. These can be customised:

pointer:
  key: "q"
  color: "#ff0000"
  pointerSize: 16
  alwaysVisible: false

Attribution

Attribution Extension

The attribution extension displays text sideways along the right edge of slides — useful for image credits and sources.

Photo by Unsplash

Attribution Usage

Install:

quarto add quarto-ext/attribution

Enable in YAML:

revealjs-plugins:
  - attribution

Add to any slide with a fenced div:

::: {.attribution}
Photo by [Author](https://example.com)
:::

The text rotates 180 degrees and sticks to the right edge of the viewport, even when resizing or toggling fullscreen.

Font Awesome

Font Awesome Extension

The fontawesome extension lets you use Font Awesome 6 icons via shortcodes.

Install:

quarto add quarto-ext/fontawesome

Solid Icons (default)

thumbs-up | heart | star | check

Brand Icons

brands github | brands r-project | brands python

With Size

size=large | size=Large | size=huge

Font Awesome Syntax

Basic usage:

{{< fa icon-name >}}

With icon group (solid, regular, brands):

{{< fa brands github >}}
{{< fa regular heart >}}

With optional size:

{{< fa star size=large >}}

Available sizes: tiny, scriptsize, footnotesize, small, normalsize, large, Large, LARGE, huge, Huge.

Font Awesome in Practice

Use icons inline with text for visual cues:

  • Source code on GitHub
  • Contact via email
  • Data visualisation
  • Data storage
  • Command line tools
  • Documentation
  • Team collaboration
  • Bug reports

Lightbox

Lightbox Extension

The lightbox extension adds click-to-zoom functionality to images. Click any image below to open it in a lightbox overlay.

Install:

quarto add quarto-ext/lightbox

Enable in YAML:

format:
  revealjs:
    lightbox: true

Lightbox Demo

Click on any image to zoom in. Press Esc or click outside to close.

Images with the same group attribute can be navigated as a gallery in the lightbox view.

Lightbox Syntax

Add .lightbox class or set lightbox: true globally:

![Caption](image.png)

Group images into a gallery:

![Image A](a.png){group="my-gallery"}
![Image B](b.png){group="my-gallery"}

Include Code Files

Include Code Files Extension

The include-code-files extension lets you include code from external source files directly into your slides.

Install:

quarto add quarto-ext/include-code-files

Enable in YAML:

filters:
  - include-code-files

Including an Entire File

Reference an external file with the include attribute on a code block:

```{.python include="script.py"}
```

The file contents replace the empty code block at render time.

Including Specific Lines

Use start-line and end-line to include a range:

```{.r include="analysis.R" start-line=5 end-line=15}
```

Including Snippets

Mark regions in your source file with comments:

# start snippet my-function
def hello(name):
    return f"Hello, {name}!"
# end snippet my-function

Then reference the snippet by name:

```{.python include="script.py" snippet="my-function"}
```

Supported comment styles: # (Python/R), // (JS/CSS), -- (Lua), <!-- --> (HTML).

Clean Theme

Clean Theme Extension

The clean theme by Grant McDermott provides a minimalist, elegant look for academic and professional presentations.

Install:

quarto add grantmcdermott/quarto-revealjs-clean

Use as the format:

format:
  clean-revealjs:
    slide-number: true

Clean Theme Features

The theme provides:

  • Minimalist design with clean typography
  • Slide numbers enabled by default
  • Left-side navigation menu
  • MathJax 4 with custom configuration
  • Long date format
  • Custom SCSS styling

To use it, change the format from revealjs to clean-revealjs. Since this presentation uses revealjs with the dark theme, the clean theme is not active here — it is an alternative format.

Reveal Header

Reveal Header Extension

The reveal-header extension adds a persistent header with text and/or a logo across all slides.

Install:

quarto add shafayetShafee/reveal-header

Look at the top of this slide — the Quarto logo and “Extensions Demo” text are rendered by this extension.

Reveal Header Configuration

Enable in YAML:

filters:
  - reveal-header
header: "My Presentation"
header-logo: https://quarto.org/quarto.png

Additional Options

# Use the document title as header text
title-as-header: true

# Use the subtitle instead
subtitle-as-header: true

# Hide header from title slide
hide-from-titleSlide: all   # or "text" or "logo"

# Make the logo a clickable link
header-logo-link: "https://example.com"

Reveal Header on This Presentation

This presentation uses the following configuration:

header: "Extensions Demo"
header-logo: https://quarto.org/quarto.png
hide-from-titleSlide: all
filters:
  - reveal-header

The header and logo appear on every slide except the title slide.
