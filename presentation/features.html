Smaller Text

The .smaller class reduces text size for content-heavy slides. This is useful when you need to fit more information on a single slide.

Feature Description
.smaller Reduces font size for the slide
.scrollable Adds a scrollbar for overflow
.center Vertically centers content
.nostretch Prevents images from stretching

Output Locations

Control where output appears relative to code using output-location:

  • fragment — output appears as a fragment (click to reveal)
  • slide — output appears on the next slide
  • column — output appears in an adjacent column
  • column-fragment — column output as a fragment
